# attention_calculations

Тестовое задание по принципам вычисления attention в нейронных сетях:

0. Проект не содержит зависимостей, отличных от библиотеки std.
0.1. В рамках проекта сразу же собирается один target, без создания core библиотек(и).

1. Сопроводительная документация:
1.1. Математический аппарат Attention как слоя нейронной сети описан в файле "Attention_theory.txt"
1.2. Результаты вызова различных реализаций Attention на моем компьютере зафиксированы в файле "Attention_results.txt"


2. Есть следующие реализации Attention:
2.1. MatMulType::NAIVE. 
	Последовательно вычисляются все матричные произведения.
2.2. MatMulType::CACHE_OPTIMIZED. 
	Матричные произведения выполняются по блокам.
	Применяется online softmax.
2.3. MatMulType::SIMD.
	Матричные произведения выполняются по блокам.
	Применяется online softmax.
	Применяется векторизация AVX.
